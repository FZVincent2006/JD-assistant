# 招聘 JD 发布助手安装说明

本安装包适用于 macOS 上的 Chrome 和 Edge，同时支持 Apple Silicon 与 Intel Mac。

## 一、加载扩展

1. 解压整个安装包，不要只移动其中某个文件。
2. Chrome 打开 `chrome://extensions`；Edge 打开 `edge://extensions`。
3. 开启“开发者模式”，点击“加载已解压的扩展程序”，选择安装包内的 `扩展` 文件夹。
4. 浏览器显示的 32 位扩展 ID 必须与 `VERSION.txt` 的 `EXTENSION_ID` 完全一致；不一致时停止安装并联系管理员。

## 二、安装飞书授权助手

1. 双击 `安装飞书授权助手.command`。
2. 终端提示后，由飞书应用管理员在这台 Mac 上输入 App Secret 并回车；输入过程不会显示字符。
3. App Secret 只保存在当前 macOS 用户的 Keychain 中，不会写入安装包或命令历史。
4. 安装完成后，完全退出并重新打开 Chrome 或 Edge。

如果 macOS 阻止首次运行，请打开“系统设置 → 隐私与安全性”，使用系统显示的“仍要打开”，然后重新双击安装入口。不要运行来源不明的解除隔离命令。本版本不需要辅助功能、屏幕录制或输入监控权限。

## 三、飞书授权与检查

1. 打开扩展侧栏，选择“飞书文档”。
2. 点击“授权飞书”，使用自己的企业飞书账号完成授权。
3. 点击“检查测试副本”，确认能够读取文档版本和两区公司数量。
4. 正式文档没有写入入口；所有测试统一使用扩展中锁定的测试副本。

## 四、日常使用

扩展通过 OpenAPI 依次写入和校验岗位 JD 与 Portfolio，不会滚动飞书页面或发送快捷键。公司名会写成根级一级标题；需要显示 `1.` 自动序号时，请在写入成功后到飞书页面手动设置。

## 五、旧版本回退

新版本验收前不要删除原来的旧版本扩展文件夹。如果新版本无法授权或检查测试副本，可在扩展管理页移除新版本，再重新“加载已解压的扩展程序”并选择旧版本目录。回退扩展不会撤销或覆盖已经写入的文档内容。
